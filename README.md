# Microserviço Contas
### Acessando ao H2
#### 1. Verificar no console o endpoint criado:
````
h2-console
````
#### 1. No navegador colocar o endpoint criado com a porta local cofigurado nas properties. Se não tiver sido configurada, ficará na 8080:
````
http://localhost:8080/h2-console/
````
